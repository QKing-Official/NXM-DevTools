# DevTools

Developer Utilities to make expansion development easier.

DevTools is a utility expansion that provides helpful tools for developers, including logging, inspection, and debugging features. It is open source and designed to simplify development workflows.

---

## Usage

Download DevTools and place it into your expansions folder to use it.  

---

## Features

- Developer utility tools  
- Advanced logging and log viewing  
- Debug and inspection helpers  
- Lightweight and easy to use  
- Free and open source
